Some Branching & Merging commands

  1. git branch {List branches (the asterisk denotes the current branch)}
  2. git branch [branch name] {Create a new branch}
  3. git checkout [branch name] {Switch to a branch}
  4. git push -u origin [branch name]
  5. git merge [branch name] {Merge a branch into the active branch}

Some Pull, Toggle branch & merge conflict commands

  1. git checkout -b [branch name] {Create a new branch and switch to it}
  2. git pull {Update local repository to the newest commit}
  3. git status {Check status}
